# acp-client (스캐폴드)

AI 에이전트 시뮬레이터(Compose Desktop) 모듈입니다.

> ⚠️ **현재 상태: 스캐폴드(미구현)**
> 빌드 설정(`build.gradle.kts`)과 디렉토리 구조만 존재하며, UI/에이전트 로직은 아직 구현되지 않았습니다.
> 서버(`acp-merchant`, `acp-psp`) 데모는 이 모듈 없이 동작합니다 — 루트 [README](../README.md)의 "빠른 시작" 참고.

## 예정 범위
- 채팅 인터페이스(ChatGPT 스타일) 및 상품 검색/장바구니/체크아웃 플로우 시각화
- 프로토콜 디버거(요청/응답 JSON 타임라인)
- 시나리오 테스트 (상품 검색 → 체크아웃 → 결제)
